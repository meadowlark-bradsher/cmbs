# IR Snapshot Docs

This folder stores generated inventory/reference snapshots of the repository and runtime surface.

## Files

- `repo_map.md` - structural map of repository components
- `integration_surface.md` - integration touchpoints and interfaces
- `runtime_capabilities.md` - runtime feature/capability view
- `state_and_events.md` - state model and event inventory

## Notes

- These files are snapshots, not the canonical specification.
- Keep source-of-truth specs in `docs-vault/`.
